# 地堡防线 · 开场动画 Gemini 提示词（手机版）

## 通用设定
**风格：** 美式卡通，PvZ + Don't Starve 描边风
**画幅：** 9:16 竖屏（手机）
**时长：** 约7~8秒
**视角：** 玩家在底部 → 怪物从上方出现向下移动
**镜头：** 固定2D，仅上下平移和缩放，无旋转

## 分镜

### 00:00~01:5 — 装载
**画面：** 庇护所内部，竖屏构图
**镜头：** 固定
**元素：**
- 红光闪烁一次
- 画面下方一个朝上的发射槽
- 从上到下一排丧尸排队站着，最前面一只自己跨进发射槽
- 舱门"咔"关上

### 01:5~03:0 — 上升
**画面：** 镜头沿竖井向上平移
**元素：**
- 快速掠过管道/线缆剪影
- 光线越来越亮
- 穿出地面瞬间全白一帧

### 03:0~05:0 — 防线
**画面：** 全白消退 → 地表防线
**镜头：** 从防线高度逐渐向后拉远
**元素：**
- 画面下方中央一个**高台**，主角站在上面，背对画面，面向前方战场
- 高台下方防线矮墙横贯底部
- 前方是向上延伸的废土斜坡
- 主角头部冒出**气泡文字**："又来？行吧。"

### 05:0~07:5 — 展开
**镜头：** 继续后拉至游戏标准视野，固定
**画面：** 完整竖向战斗场景
- **底部：** 高台 + 主角 + 防线矮墙，两侧有炮台/陷阱位
- **中部：** 斜坡战场，散落瓦砾枯草
- **顶部：** 倒塌城市剪影 + 灰黄天空
- **左上 HUD：** 弹药槽（丧尸头像）、生命值、关卡名

定住 → 浮现标题 "第一章 · 晋升"
