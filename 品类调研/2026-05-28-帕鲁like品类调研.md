## 1. 品类定义

**一句话：** 玩家通过探索、捕捉/驯服、养成和战斗来收集各类生物，并利用它们进行战斗、生产或建造的玩法品类。

**品类谱系：**
```
宝可梦类（回合制RPG战斗）
  ├── 传统宝可梦（NS）
  ├── 类宝可梦独立游戏（Temtem / Cassette Beasts / Nexomon）
  └── IP衍生（数码宝贝 / 洛克王国）
帕鲁like（开放世界生存+驯服）
  ├── Palworld（标杆，Steam历史CCU第一）
  └── 帕鲁like手游（Dawnlands / Palmon: Survival）
收集+合成变体
  └── 召唤与合成（消除+收集）
```

## 5. 融合评估

### 适配度评分：★★★☆☆（三星）

正向因素：
- 养成框架天然吻合，生物收集可视为养成横向扩展
- 手机端收集品类已验证（洛克王国¥6,304万/10天）
- 帕鲁like的战斗辅助模式与塔防可互补

负向因素：
- 核心乐趣冲突：即时操作反馈 vs 长期积累策略
- 竖屏限制探索体验
- 5-15人团队做生物收集美术压力大
- 宝可梦IP垄断，帕鲁已有头部
