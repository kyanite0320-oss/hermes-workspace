4. 深度拆解1：无尽冬日 — 竖屏商业化标杆

核心数据

  • 畅销榜Top10，4月周年庆单日流水#2
  • 竖屏SLG品类最成功的商业化产品
  • 用户画像与塔防射击高度重叠（竖屏重度手游用户）

核心玩法循环

进游戏收菜 → 建筑升级 → 英雄编队 PvE → 世界地图采集/战斗 → 养成升级 → 联盟社交 → 限时活动

核心设计：Bait n' Switch 漏斗

  • 前90分钟 = Frozen City式的轻量闲置经营（建床铺/猎人小屋/锯木厂）—— 完美匹配买量素材
  • 90分钟后 = 世界地图4X展开，变成重度SLG
  • 首次实现"4X SLG"作为闲置经营后续阶段的无缝过渡

关键设计：7条养成线互相独立，每一条都有R→SR→SSR→UR品质阶梯

商业化三层漏斗

层设计
第一层冰霜之星（Frost Star） — 独立付费货币，$4.99起。用于永久性账户升级
第二层月卡+通行证多轨 — 月卡$5 + 精英通行证 + 活动通行证多个并行
第三层分层礼包体系 — 弹出式限时礼包、英雄Gacha、节日礼包

二级货币设计精妙

  • 免费钻石：每日可积累约80,000（≈€95价值），但只能花在"过程性资源"上
  • 付费冰霜之星：仅直充，用于永久队列解锁/功能解锁
  • 免费送很多但关键点必须付费

活动运营密度

每天至少2-3个活动同时在线，形成"每天上线都有事做"。

5. 深度拆解2：搜打撤（超自然行动组）— 新玩法模式空白

超自然行动组关键数据

  • 1000万DAU，畅销榜Top4
  • 环比增长80%
  • 开发成本传闻800万，ROI极高
  • 轻度化搜打撤的代表作（比塔科夫/三角洲更容易上手）

搜打撤核心循环

准备（选装备/组队）→ 进入地图 → 搜索资源（破机关/开宝箱） → 战斗（打粽子/超自然生物）→ 决策（走还是留） → 撤离（到撤离点读条）→ 结算（资源转化/养成）

搜打撤品类设计本质

打破了传统"局内/局外独立"的设计范式： - 局内资源可带到局外 → 再转化为下次局的资本 - 创造永不结束的进度感 - 高风险高回报的过山车式情绪体验 = 核心乐趣

🔥 塔防版搜打撤 融合方案

为什么这可能是你们的制胜点？ — 塔防版搜打撤目前市场上无竞品，是真正的品类融合空白点。

搜打撤原版→ 塔防版映射
自由搜索资源地图分支路线中点触拾取
射击战斗炮台自动射击 + 主动技能
到达撤离点防守波次完成后撤离
装备带入/带出炮台/角色/道具养成

塔防版搜打撤玩法流程

准备（选英雄/配炮台/带保险）→ 进图 → 选路线搜资源（岔路选择决定收益） → 定点防守（3-5波塔防战）→ 决策（立即撤离 vs 继续深入高风险区） → 防守撤离读条 → 结算

商业化挂钩

付费点设计
保险系统（最关键）死亡→免费丢失部分资源 vs 付费保险→无损
英雄抽取+炮台养成Gacha + 升星
入场券/体力免费体力有限 + 付费入场券

MVP落地（5-15人团队 2~3个月）

  • 1张地图（3条岔路 + 1个防守点）
  • 1个角色 + 2种炮台
  • 3种怪物 + 3波防守
  • 保险券（1种）
  • 基础养成（角色升级+炮台升级）
  • 撤离决策（立即撤/继续深入）

7. 落地路线图

第一阶段（即刻-1个月）：商业化骨架

  1. 赛季通行证三轨制（免费/$4.99/$9.99）
  2. VIP等级系统 + 月费钻石激活
  3. 付费解锁第2个炮台/技能槽
  4. 活动日历规划（每周小活动+每月大活动）

第二阶段（1-3个月）：差异化玩法 + 内容扩展

  1. 搜打撤塔防MVP — 核心循环验证
    • 1张地图、3条岔路、1个防守点
    • 保险系统 + 基础养成
  2. 限时活动轮换（无尽模式/Boss Rush）
  3. 二级货币体系

第三阶段（3-6个月）：深度扩展

  1. 宠物/伙伴系统
  2. IP联动首发
  3. 搜打撤多地图扩展
  4. 皮肤+特效变现
